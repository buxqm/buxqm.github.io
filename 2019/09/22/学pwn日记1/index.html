day1

1.初次尝试reversing

使用gdb反汇编二进制程序gdb license_1

输入set disassembly-flavor intel 把gdb配置成intel语法，在intel语法中，汇编指令遵循下面这种形式：operation <destination><source> 可以更直观的观察寄存器，内存地址和命令。

输入 disassemble main 显示汇编指令，这里就不展示代码。

在程序开始时设置断点开始调试break * main 首先介绍使用的指令si/ni si会执行一行程序代码，如果有函数调用则进入该函数，ni则不会进入内部。

解题思路：1.strcmp的结果放在eax寄存器中，也就是rax的低位,如果为0，则strcmp返回正确。

2.用于反转二进制文件的一些常见工具和技术

strings license_1提取出文件中的字符

linux 内存管理（text,rodata,data,bss,stack,&heap）

linux内存管理简介

获取rodata的地址（使用objdump -x license_1| less），然后dbg x/s命令显示字符串，可以直接看到C语眼代码中定义的常量。

strace ./license_1 用strace运行文件，目前我只知道可以显示输出的字符串。

itrace ./license_1目前我只知道可以显示printf和strcmp的语句。

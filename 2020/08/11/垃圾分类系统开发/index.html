数据集处理

将自定数据集转为coco格式

电脑环境：win10,anadonda

coco数据集一般用于目标检测和实例分割，它使用边界框（bounding boxes）标注物体，它的文件结构如下。

在train.json中和val.json中存储与标注信息、图像文件相关的信息，其详细信息可以看 PaddlePaddle数据集格式说明

将自定义数据集转为COCO格式复杂点在于创建“annotations”列表，其他的都是可以直接利用你的数据创建。

“annotations”需要使用pycocotools第三方库生成：下载地址。

如果你在Window10系统下如果你直接使用

pip install git+git://github.com/waspinator/coco.git@2.1.0安装pycococtools的话会出现报错

error: Microsoft Visual C++ 14.0 is required.

很明显，需要Visual C++ 14.0环境，在网上找了一圈，需要安装visual c++ build tools

下载地址：https://blog.csdn.net/u012247418/article/details/82314129

安装完后在开始菜单进入”Visual C++ 2015 x64 Native Build Tools Command Prompt “命令行。

步骤一：激活虚拟环境activate env_name

这里我蒙蔽了一下，我不知道虚拟环境名称是啥，后面去了解了一下虚拟环境。

博客地址：https://www.cnblogs.com/yuehouse/p/10239195.html

简单来说虚拟环境就是利用docker把环境内容放在容器中，然后运行，docker容器是相互隔离的，这样很方便管理。于是我使用conda env list查看了一下我的虚拟环境，果然只有在安装anaconda时创建的base环境（太菜了）接下来就执行activate base

步骤二：安装pycocotools

直接执行pip install git+git://github.com/waspinator/coco.git@2.1.0在anaconda基础镜像中安装pycocotools库。
